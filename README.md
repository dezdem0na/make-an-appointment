# make-an-appointment
Clone git project 
```sh
$ git clone git@github.com:dezdem0na/make-an-appointment.git 
```
Install dependencies
```sh
$ pip install -r requirements.txt
```

Run devserver
```sh
$ manage.py runserver
```

Run tests
```sh
$ coverage run manage.py test
```

Run docker 
```sh
$ docker-compose up
```
